Sourcemation Repository

How To Use It

To access charts from our repository, please add it to your Helm client:

helm repo add sourcemation https://sourcemation.github.io/charts/

Then, update your local repository cache:

helm repo update

You can search for available charts using:

helm search repo sourcemation
